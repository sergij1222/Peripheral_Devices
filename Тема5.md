# Типи та характеристики SSD-накопичувачів

## 1. SATA SSD

![alt](https://www.samsung-online.com.ua/uploads/shop/products/additional/5a2ad78f839daaabdd4c67dca5c486c5.5_V-NAND_4bit_MLC_SATA_III_(MZ-77Q1T0BW)1.jpg)

Результат:

**SATA SSD** — це твердотільний накопичувач, який підключається через інтерфейс SATA III.

### Основні характеристики:
- Інтерфейс: SATA III (6 Гбіт/с)
- Максимальна швидкість читання: ~550 МБ/с
- Максимальна швидкість запису: ~520 МБ/с
- Форм-фактор: 2.5"
- Доступна ціна

### Переваги:
- Простота встановлення
- Сумісність зі старими ПК
- Низька вартість

---

## 2. M.2 SSD

![alt](https://memory.net.ua/media/wysiwyg/SSDM2_01.jpg)
**M.2 SSD** — компактний накопичувач, який підключається безпосередньо до материнської плати.

### Основні характеристики:
- Форм-фактор: M.2
- Інтерфейс: SATA або PCIe
- Компактні розміри
- Підвищена швидкість передачі даних

---

## 3. NVMe SSD


![alt](https://dmassets.micron.com/is/image/microntechnology/product-2400-m2-group:3-2-all-others?ts=1733767550197&dpr=off)
**NVMe SSD** використовує інтерфейс PCIe та протокол NVMe.

### Основні характеристики:
- Інтерфейс: PCIe 3.0 / 4.0 / 5.0
- Швидкість читання: до 7000+ МБ/с
- Низька затримка
- Висока продуктивність

### Переваги:
- Ідеально для ігор та роботи з відео
- Максимальна швидкість

---

# Порівняльна таблиця

| Тип SSD   | Інтерфейс | Швидкість читання | Форм-фактор |
|------------|------------|-------------------|-------------|
| SATA       | SATA III   | ~550 МБ/с         | 2.5"        |
| M.2 SATA   | SATA       | ~550 МБ/с         | M.2         |
| NVMe       | PCIe       | 2000–7000+ МБ/с   | M.2         |

# Скрипт для відображення технічної інформації про накопичувач

## Додаткові команди Linux для перегляду інформації про накопичувач

### 1. Перегляд усіх дисків та розділів
```bash
sudo fdisk -l
```
Показує список усіх підключених накопичувачів, їх розміри, типи розділів та файлові системи.

### 2. Детальна інформація про диск (швидкість, кеш тощо)
```bash
sudo hdparm -I /dev/sda
```
Виводить технічні характеристики накопичувача:
- модель
- серійний номер
- підтримувані режими
- швидкість передачі даних

(замість `/dev/sda` вкажіть свій диск, наприклад `/dev/nvme0n1`)

### 3. Перевірка SMART-стану диска
```bash
sudo smartctl -a /dev/sda
```
Показує:
- стан здоров’я накопичувача
- кількість помилок
- температуру
- кількість перезаписів (для SSD)

Для роботи smartctl потрібно встановити пакет(АЛЕ В МЕНЕ НЕ ВИЙШЛО ВСТАНОВИТИ ЧОМУСЬ ):
```bash
sudo apt install smartmontools
```

---

### 4. Перегляд інформації про блочні пристрої
```bash
lsblk -o NAME,SIZE,TYPE,MODEL,FSTYPE
```
Зручно показує структуру дисків і розділів у вигляді дерева.
